CLASTR - About

Publication
Robin, T., Capes-Davis, A. & Bairoch, A. CLASTR: the Cellosaurus STR Similarity Search Tool - A Precious Help for Cell Line Authentication. International Journal of Cancer (2019).
PubMed: 31444973  DOI: 10.1002/IJC.32639
Credits

Developed by: Thibault Robin; University of Geneva and SIB-Swiss Institute of Bioinformatics

  • Frontend written in JavaScript with the jQuery and jQuery UI libraries
  • Parsing of plain text file (CSV, TSV, TXT) formats powered by Papa Parse
  • Parsing of Excel file (XLS, XLSX) formats powered by SheetJS js-xlsx
  • Header pattern made with trianglify.io
  • Background tile made by Shiva from toptal.com
  • File icons made by Smashicons from flaticon.com
Changelog
Version 1.4.4
  • Improve the compatibility of mouse STR marker names
Version 1.4.3
  • Add a descriptive comment above the result table and improve the alert messages
Version 1.4.2
  • Fix the Mouse STR 9-2 marker being sometimes included in the scoring
Version 1.4.1
  • Remove the explicit species name in the result tables for readability concerns
Version 1.4.0
  • Add the possibility to search mouse or dog STR profiles
Version 1.3.0
  • Add a filter to limit matches to a minimum number of common STR markers
  • Change the color of alleles to gray when the query does not have allele data for a given STR marker
Version 1.2.0
  • Rename the tool to CLASTR (Cell Line Authentication using STR)
  • Add the Cellosaurus entry name in the page title when loading an entry
Version 1.1.1
  • Add the NCEB-1 (CVCL_1875) hybrid cell line as its STR profile is normal
  • Fix the secondary accession being taken instead of the primary one in some cases
Version 1.1.0
  • Make the Excel XLSX format available in both the web user interface and the RESTful API
  • Remove the PDF format as it was unsatisfactory and can be easily exported from XLSX files
  • Add the possibility to change the name and format of the batch query file in the load file dialog
  • Add a progress bar to the load and export file dialogs
  • Rename the haplotypes field as profiles in the JSON response of the RESTful API
Version 1.0.1
  • Fix the parameters not properly actualizing when loading successive files
Version 1.0.0
  • Full release on the ExPASy server
  • Integration in the Cellosaurus database
